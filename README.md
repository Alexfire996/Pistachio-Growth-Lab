# Pistachios Growth Lab

Official static website for Pistachios Growth Lab.

Pistachios Growth Lab helps young, high-potential brands clarify their growth strategy across acquisition, positioning, onboarding, activation, and practical go-to-market execution. The site is bilingual, with English and Chinese content for international and Chinese clients.

## Website

This is a lightweight static site built with plain HTML, CSS, and JavaScript. It is ready to deploy on Vercel, Netlify, GitHub Pages, or any static hosting service.

## Pages And Files

- `index.html` - main website page
- `styles.css` - visual design, layout, responsive styles, and mobile/iPad support
- `script.js` - language switching and bilingual website copy
- `pgl-logo.png` - original logo image
- `pgl-logo-transparent.png` - transparent logo used in the header
- `cta-variations.html` - separate CTA concept explorations

## Local Preview

Open `index.html` directly in a browser.

For a local server preview:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Deploy On Vercel

1. Import this GitHub repository into Vercel
2. Choose the branch you want to deploy
3. Keep the framework preset as `Other`
4. Leave build command empty
5. Leave output directory empty
6. Deploy

Because this is a static HTML/CSS/JS site, no build step is required.
